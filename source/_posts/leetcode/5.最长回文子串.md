> 最长回文子串

## 什么是回文串

比如单词：level, noon，特性就是第 n 位，和第 -n 位相等。

<!--more-->
```shell
func isPalindromic(str string) bool {
	left := 0
	right := len(str) - 1
	for left < right {
		if str[left] == str[right] {
			left++
			right--
		} else {
			return false
		}
	}
	return true
}
```

## 解1

> 暴力求解

```shell
func longestPalindrome(s string) string {
	if len(s) == 1 {
		return s
	}
	var r string
	for i := 0; i < len(s); i++ {
		for j := i + 1; j < len(s)+1; j++ {
			subStr := s[i:j] //前闭后开
			if isPalindromic(subStr) && len(subStr) > len(r) {
				r = subStr
			}
		}
	}
	return r
}
```
