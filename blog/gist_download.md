> 个人使用。我会在 gist 写 blog，然后再使用工具下载为本地相对引用的 markdown文件。

![](../images/8da20eebd2b8879bdbedfadaa08678f7.jpeg)


## 好处

- gist 写 markdown 很舒服，浏览器就搞定了
<!--more-->
- gist 可以直接粘贴剪贴板的图片. 如果是本地编辑器，需要保存，然后想对引用，比较繁杂
- gist 主要是备份


## 使用

```
btl gist -u <gist_id>
```
