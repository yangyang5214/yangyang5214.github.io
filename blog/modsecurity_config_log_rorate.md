- set modsecurity with logrotate
- reload apachectl config(use graceful)

<!--more-->
```
vim /etc/logrotate.d/modsec_audit

/var/log/modsec_audit.log {
   postrotate 
      apachectl -k graceful > /dev/null 2>/dev/null || true
   endscript
}
```
